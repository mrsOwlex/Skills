# AGENTS.md

Guidance for agents working in this repository.

- Treat the repository root as human-facing project documentation.
- Treat `skills/<skill-name>/` as agent-facing skill content.
- Keep each skill folder self-contained and minimal: `SKILL.md`, optional `agents/openai.yaml`, and optional `scripts/`, `references/`, or `assets/`.
- Do not place generated review reports, temporary test repositories, or large artifacts in this repository.
- Validate changed skills before committing when the validator is available.
